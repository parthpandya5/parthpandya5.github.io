CrisisComm: LLM for Post-Disaster Aid Acquisition

The purpose of this project is to familiarize myself with fine tuning LLMs using a pre-trained, open-source model. I also learned how to communicate between the model and the vector database using a Langchain environment. This is a web application that utilizes Streamlit's python library to send user-inputted prompts on the client-side to the local server-side where key words in the prompt are matched against the vector database, and then use the pre-trained model alongside with the data to deliver a contextually appropriate response to the user. I developed this project primarily using Python, Langchain, Mistral, Llama3 and Chroma DB. Moreover, I also published a chapter for this project in a book thats on Amazon.
